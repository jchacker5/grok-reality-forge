# Grok Reality Forge

Grok Reality Forge is a visionOS-native app for Apple Vision Pro that turns text or voice prompts into immersive spatial worlds using the xAI image API. Create, explore, and iterate on panoramic environments anchored to your space.

## Features
- Voice and text prompt input
- High-fidelity panoramic generation via xAI API
- Spatial anchoring and immersive exploration
- Real-time iteration and refinement
- Starter worlds to jump in immediately

## Setup
1. Open the project in Xcode.
2. Provide your xAI API key via `.env` (not committed) or the in-app settings.
3. Build and run on Apple Vision Pro or the Vision Pro simulator.
